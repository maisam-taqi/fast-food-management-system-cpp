# Fast Food Management System (C++)

A console-based Fast Food Management System developed in C++ as a Programming Fundamentals semester project.

## Features

- Secure Login System
- Customer Management
- Employee Management
- File Handling
- Password Protection
- CRUD Operations

## Technologies Used

- C++
- File Handling (fstream)
- Functions
- Loops
- Switch Case

## How to Run

1. Open the project in Dev-C++ or VS Code.
2. Compile the source code using C++ compiler.
3. Run the program.
